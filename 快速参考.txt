# 快速参考 - OpenClaw 图片生成

> ⚠️ %USERPROFILE% = 你的 Windows 用户名

## 首次使用（一次性）

1. 复制 Skill：`gemini-image-SKILL.md` → `%USERPROFILE%\.openclaw\workspace\skills\gemini-image\SKILL.md`
2. 复制脚本：`wait_download_cache.py` → `%USERPROFILE%\.openclaw\workspace\wait_download_cache.py`
3. 创建 Chrome 配置目录
4. 启动 Chrome（含调试端口）
5. 登录 `https://gemini.google.com`

## 关键路径

| 用途 | 路径 |
|------|------|
| Chrome 配置 | `%USERPROFILE%\AppData\Local\Google\Chrome\OpenClaw` |
| 图片保存 | `%USERPROFILE%\.openclaw\media\` |
| 下载监控脚本 | `%USERPROFILE%\.openclaw\workspace\wait_download_cache.py` |
| Skill | `%USERPROFILE%\.openclaw\workspace\skills\gemini-image\SKILL.md` |

## 提示词前缀（必须）

`Generate Image ` （英文描述）

## 常见错误

- "工具按钮是灰的" → Chrome 未登录，先登录
- "下载没反应" → 等 1-3 分钟，大文件下载慢
- "图片发不出去" → 检查 accountId 是否正确
